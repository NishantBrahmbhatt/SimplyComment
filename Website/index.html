How it works

Setup

One-time setup.
Comment forever after.

SimplyComment needs your OpenAI API key to generate comments. Add it once in Settings — stored locally in your browser — and you're ready to go.

  1. 1

    Add SimplyComment to Chrome

    Install the extension from the Chrome Web Store. Pin it to your toolbar so it's easy to open on LinkedIn.

  2. 2

    Add your API key in Settings

    Open SimplyComment → Settings and paste your OpenAI API key (starts with sk-). Generation won't work until this is saved. How to find your API key

  3. 3

    Build your voice profile

    In Settings, add a few real comments and posts you've written. The more authentic samples you give it, the better SimplyComment matches your tone.

  4. 4

    Generate your first comment

    On LinkedIn, open SimplyComment, paste the post you want to reply to, choose Polish Draft or Help Me Ideate, and copy your comment. Done.

SimplyComment Privacy Policy

Last updated: April 28, 2026

SimplyComment is a browser extension that helps users generate short comments from text they manually paste into the extension. This policy explains what data is processed, where it is stored, and how it is used.

1) Data We Process

SimplyComment processes the following data provided by the user:

  • OpenAI API key
  • Post text pasted into the extension
  • User angle/draft text
  • Optional profile and voice settings (name, role, tone notes, sample comments, sample posts)
  • Generated output text

2) How Data Is Used

  • To generate comment suggestions requested by the user
  • To preserve user settings and drafts between sessions
  • To personalize output tone based on optional user-provided examples

3) Where Data Is Stored

  • chrome.storage.sync: API key and profile/settings data
  • chrome.storage.local: temporary draft data (for popup persistence)

Data is stored by the browser extension storage APIs on the user’s browser profile (and sync account if sync is enabled).

4) Third-Party Services

SimplyComment sends user prompt content to OpenAI to generate comment suggestions. Requests are sent only to:

  • https://api.openai.com

OpenAI processes request data according to OpenAI’s own terms and privacy policies.

5) What SimplyComment Does Not Do

  • Does not auto-post content on behalf of users
  • Does not sell personal data
  • Does not use data for advertising
  • Does not scrape website content automatically for this paste-based workflow

6) Data Sharing

Data is shared only as required for core functionality (i.e., sending prompts to OpenAI). We do not sell, rent, or broker user data to third parties.

7) Security

We use browser extension storage APIs and limit permissions to those needed for core functionality. However, no method of storage or transmission is completely secure.

8) User Control

Users can:

  • Edit or remove stored settings inside the extension UI
  • Uninstall the extension to remove extension-managed local data
  • Revoke/rotate their OpenAI API key at OpenAI

9) Children

SimplyComment is not directed to children under 13, and we do not knowingly collect personal data from children.

10) Changes to This Policy

This Privacy Policy may be updated from time to time. Updates will be reflected by revising the “Last updated” date on this page.
